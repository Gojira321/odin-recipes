Piri-Piri Chicken

  • Some round bird(Chicken)
  • Some spicy peppers(Chillis)
  • Some sour oranges(Lemons)
  • Some red powder(smoked Parika)
  • Some red paste(tomato Paste)
  • Some green stuff(thyme)
  • Some smelly onions(garlic)
  • Some not smelly garlic(shallots)
  1. In a morter grind the sour spicy peppers, the smelly onions, the not smelly garlic
  2. Ready the red paste in a pan and add to the mix
  3. Also add some olive oil and lemon juice to the mix
  4. Prepare the chicken and add the marinate. Set the chicken in the fridge to marinate from 24-48 hours
